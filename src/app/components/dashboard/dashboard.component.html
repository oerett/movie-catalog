Welcome to the Movie Catalog

This is a movie catalog application where you can explore and search for your favorite movies. The application pulls data from a powerful movie database API (OMDb API) that provides detailed information about movies, including posters, release years, ratings, and more.

Use the search functionality to find movies by name, and click on their posters to learn more details about each film. The catalog includes top matches based on your search criteria, making it easy to discover new and popular movies!
